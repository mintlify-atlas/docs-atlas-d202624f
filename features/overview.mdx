## Search Modes

<Card title="17 Specialized Search Modes" icon="magnifying-glass" href="/features/search-modes">
  Choose from Web, Chat, X, Stocks, Code, Academic, Extreme, Reddit, GitHub, Crypto, Prediction, YouTube, Spotify, Connectors, Memory, Voice, and XQL modes for different research needs.
</Card>

Scira adapts its search strategy based on your query. Whether you need live social media trends, academic papers, financial data, or code examples, there's a mode optimized for it.

**Examples:**
- **Web mode**: General research questions with comprehensive web coverage
- **Academic mode**: Research papers and scholarly sources with citations
- **Extreme mode**: Deep multi-step research with autonomous planning
- **X mode**: Real-time social media discussions and trending topics
- **Stocks mode**: Financial data with interactive charts and SEC filings

[Explore all search modes →](/features/search-modes)

## Integrated Tools

<Card title="28 Integrated Tools" icon="wrench" href="/features/tools">
  Access web search, financial data, code execution, weather, maps, media search, and more—all through natural language.
</Card>

Scira's tools are automatically invoked based on your query. You don't need to manually select tools—the AI agent determines which tools to use and how to combine them.

**Tool Categories:**
- **Search & Retrieval**: Multi-provider web search, academic search, Reddit, X, YouTube, GitHub
- **Financial & Market Data**: Stock charts, crypto data, currency conversion, prediction markets
- **Location & Travel**: Weather forecasts, geocoding, flight tracking
- **Media & Entertainment**: Movie/TV search, trending content
- **Productivity & Utilities**: Code interpreter, translation, file query, connectors

## Advanced Features

### Lookouts

<Card title="Scheduled Research Agents" icon="calendar" href="/features/lookouts">
  Monitor topics automatically with recurring research agents that email you updates.
</Card>

Lookouts let you schedule research queries to run automatically on a recurring basis (daily, weekly, monthly, yearly, or one-time). Each run creates a new chat with full research results and sends you an email summary.

**Use cases:**
- Track competitor mentions and market trends
- Monitor research topics for new developments
- Get daily briefings on specific industries
- Track regulatory or policy changes

[Learn about Lookouts →](/features/lookouts)

### Custom Instructions

Personalize how Scira responds by adding custom instructions that apply to all your conversations. Define your preferences for tone, format, depth, or domain-specific requirements.

### File Upload & Query

Upload PDFs, CSVs, DOCX, and Excel files to search and query their contents. Scira uses Cohere embeddings and reranking for semantic search over your documents.

### Connected Apps

Search across Google Drive, Notion, and OneDrive with the Connectors mode (Pro feature). Your personal knowledge base integrated directly into research workflows.
