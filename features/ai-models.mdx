Scira supports 100+ AI models from major providers, giving you access to the latest and most powerful language models. The platform automatically selects the best model for your task, or you can manually choose based on your specific needs.

## Model Selection

**Automatic selection** (default): Scira analyzes your query and selects the optimal model based on:
- Task complexity (reasoning, generation, analysis)
- Response time requirements
- Cost efficiency
- Specialized capabilities needed

**Manual selection**: Choose a specific model when you have requirements for:
- Specific model capabilities (vision, function calling, extended context)
- Consistency across sessions
- Cost optimization
- Provider preferences

## Provider Configuration

Scira uses the Vercel AI SDK for model integration. Supported providers are configured via:

```typescript
// From package.json dependencies:
"@ai-sdk/anthropic": "3.0.2"      // Claude models
"@ai-sdk/cohere": "3.0.2"         // Command models
"@ai-sdk/google": "3.0.2"         // Gemini models
"@ai-sdk/groq": "3.0.2"           // Groq-hosted models
"@ai-sdk/mistral": "3.0.2"        // Mistral models
"@ai-sdk/openai": "3.0.2"         // GPT models
"@ai-sdk/openai-compatible": "2.0.2" // OpenAI-compatible APIs
"@ai-sdk/xai": "3.0.3"            // Grok models
"ai": "6.0.5"                      // Vercel AI SDK core
```

## Performance Considerations

**Latency**: Flash/Instant/Fast variants offer lower latency
- Gemini Flash: ~200-500ms
- GPT 5.1 Instant: ~300-600ms
- Grok 4 Fast: ~400-700ms

**Throughput**: Smaller models can handle higher request volumes
- Ministral 3B-14B: High throughput
- Qwen 3 4B-32B: Scalable
- Standard variants: Moderate throughput

**Cost**: Varies by provider and model size
- Open models (Qwen, Mistral): Self-host cost only
- Mini/Small variants: Lower API costs
- Flagship models: Premium pricing
